Services

Mobile Frontend

O mobile frontend foca na criação da interface e experiência do usuário em apps. Utilizando Flutter, React Native ou SwiftUI, garante layouts responsivos, animações fluidas e interações intuitivas, além de consumir APIs e otimizar o desempenho.

Frontend Development

O desenvolvimento frontend foca na interface e experiência do usuário em aplicações web. Utilizando HTML para estrutura, CSS para estilização e JavaScript para interatividade, os desenvolvedores criam layouts responsivos e dinâmicos.

Backend Development

O desenvolvimento backend envolve a lógica e estrutura de aplicações, utilizando linguagens como Java, Dart, Python e Lua. Java é robusto para sistemas corporativos, Dart é versátil, Python é simples e poderoso, e Lua é leve, sendo usado em jogos e scripts.

Projects

ServBlu

O ServBlu é um aplicativo desenvolvido em Flutter que conecta usuários a prestadores de serviços. Os usuários podem se cadastrar, buscar serviços e pagar via Pix ou pessoalmente. Prestadores podem criar e publicar anúncios, enquanto o app oferece notificações e personalização de perfil, proporcionando uma experiência prática e intuitiva.

EuroMusic

O EuroMusic é um app desenvolvido em Flutter, voltado para fãs de flashbacks dos anos 80, 90 e 2000. Com diversas playlists e centenas de músicas, oferece opções de busca e uma tela de favoritos. Sem necessidade de cadastro, o app facilita o acesso ao público mais velho, proporcionando uma experiência nostálgica e intuitiva.

TriviaWorld

Este site, desenvolvido com HTML, CSS e JavaScript, oferece jogos de perguntas rápidas em três temas: geral, filmes e futebol, cada um com 10 perguntas aleatórias. Os usuários acumulam pontos durante o jogo, e o cadastro é necessário para registrar os resultados. A interface é interativa e proporciona uma experiência divertida e desafiadora para os amantes de quiz.
